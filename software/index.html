pyJac

Tool for generating C and CUDA source code for evaluating source terms and analytical Jacobian matrices for chemical kinetics ODE systems.

pyJac-v2

Tool for generating OpenCL source code for evaluating source terms and sparse analytical Jacobian matrices for chemical kinetics ODE systems.

pyMARS

Python-based Model Automatic Reduction Software. Under-development tool for automatically reducing chemical kinetic models.

MARS

Model Automatic Reduction Software: mature, but frozen, software for reducing chemical kinetic models using multiple reduction stages. Please contact for access.

PyKED

PyKED is a Python-based software package for validating and interacting with ChemKED (Chemical Kinetics Experimental Data format) files that describe fundamental experimental measurements of combustion phenomena.

PyTeCK

Software for automatically validating chemical kinetic models against experimental data described by ChemKED files; produces quantitative measure of model performance.

irrev-mech

A Python utility that converts Chemkin-format chemical kinetic models with reversible reactions into those with only irreversible reactions.
