Hướng dẫn submit kết quả lên hệ thống chấp MS-COCO

Hiện nay MS-COCO có mở hai hệ thống nộp đánh giá kết quả trên hai tập dữ liệu:

  • Đánh giá trên test-dev
  • Đánh giá trên test-challenge

Thường chúng ta chỉ đánh giá trên test-dev, tập test-challenge chỉ được submit trong khoảng thời gian giới hạn được quy định.

Dưới đây là hướng dẫn submit MS-COCO:

Bước 1:

Tạo tài khoản ở trang https://competitions.codalab.org/competitions/20794

Bước 2:

Tải bộ dữ liệu Test-dev 2017 tại đây: https://cocodataset.org/#download

Lưu ý tải 2 file:

  • 2017 Test images (41K/6GB) là file hình ảnh
  • 2017 Testing Image info (1MB) là file JSON chứa các file name của bộ test-dev khớp với trên server codalab.

Bước 3 (áp dụng với toolbox mmdetection):

Sau khi tải về hai file ở Bước 2, ta giải nén ra được 2 folder: test2017 và annotations

Sau khi huấn luyện xong mô hình, chúng ta thay đổi trong config chỗ test là đường dẫn của folder test2017 file json image_info_test-dev2017.json

Bước 4:

Chạy lệnh sau để xuất ra file json

bash ./tools/dist_test.sh [file config] \
[checkpoint] [num gpus] \
--format-only --options "jsonfile_prefix=./COCO_test-dev_results"

Ví dụ như sau:

bash ./tools/dist_test.sh data_train/cascade_rcnn_r101_fpn_20e_coco.py \
work_dirs/cascade_rcnn_r101_fpn_20e_coco/epoch_24.pth 3 \
--format-only --options "jsonfile_prefix=./data_train/COCO_test-dev_results"

Bước 5

Sau khi chạy test xong sẽ xuất ra file json COCO_test-dev_results.json, nén lại một file zip đặt là submission.zip, sau đó up file zip này lên hệ thống.

Chọn phần test-dev-2019(bbox) trên hệ thống

Điền thông tin đầy đủ, sau đó bấm submit để up file zip lên.
